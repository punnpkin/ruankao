# 13 数据流图

## 1. 基本概念

| 元素     | 说明                                 | 图元 |
| -------- | ------------------------------------ | ---- |
| 数据流   | 表示数据的流向                       |      |
| 加工     | 描述了输入数据流到输出数据流的变化   |      |
| 数据存储 | 表示暂时存储的数据，每个文件都有名字 |      |
| 外部实体 | 存在于软件系统外的人员或者组织       |      |



## 2. 数据字典

| 符号                  | 含义     | 说明                              |
| --------------------- | -------- | --------------------------------- |
| =                     | 被定义为 |                                   |
| +                     | 与       | x=a+b，x由a和b组成                |
| [...,...]或[...\|...] | 或       | x=[a,b]，x由a或b组成              |
| {...}                 | 重复     | x={a}，x包含0个或多个a            |
| (...)                 | 可选     | x=(a)，x中可能包含a，也可能不包含 |
